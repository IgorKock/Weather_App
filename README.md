## Português

### Sobre
App de previsão do tempo feito com Python e a API da OpenWeatherMap.

### Requisitos
Precisa do Python 3.13.2 or suprior, da API da OpenWeatherMap e dos seguintes pacotes do PIP: requests, tkinter (precisa do messagebox, que já vem incluso no tkinter).

### Como Compliar
Clone este repositório, adicione ele na sua IDE (ou não se preferir algum editor de texto), instale usando o PIP: requests, tkinter (messagebox já vem incluso), adicione a API da OpenWeatherMap e então compile o código.

## English

### About
Simple Weather forecast app made with Python and OpenWeatherMap's API.

### Requirements
Needs Python 3.13.2 or above, API from OpenWeatherMap and the following packages from PIP: requests, tkinter (also needs messagebox, which is included on tkinter).

### How to Compile
Clone this repository, add it to your IDE (or not if you want to use a text editor), install with PIP: requests, tkinter (messagebox is included), add the OpenWeatherMap API and then compile the code.
